# Quiz: Królowie Polski — koronacje, dynastia, kontekst

## Cel quizu

Nauka dat koronacji wszystkich królów Polski **z bogatym kontekstem historycznym**, który buduje trwałe skojarzenia — nie samo zakuwanie dat, ale osadzenie każdego króla w sieci powiązań: dynastia, poprzednik, następca, kluczowe wydarzenia panowania.

**Grupa docelowa:** klasa 6 SP, uczeń zainteresowany historią.

## Kontekst dydaktyczny — co wyświetlać przy każdym królu

Aby budować skojarzenia (elaborative encoding), każdy władca powinien mieć przypisane pola kontekstowe. Wyświetlamy je **po odpowiedzi** — zarówno po poprawnej, jak i po błędnej — jako „kartę wiedzy".

### Pola danych na władcę

```
{
  name:          "Władysław II Jagiełło",
  coronation:    "4 marca 1386",
  place:         "Kraków",
  dynasty:       "Jagiellonowie",
  reign:         "1386–1434",
  predecessor:   "Jadwiga Andegaweńska",
  successor:     "Władysław III Warneńczyk",
  keyEvents: [
    "Unia w Krewie (1385) — połączenie Polski i Litwy",
    "Bitwa pod Grunwaldem (1410) — klęska Krzyżaków",
    "Założenie Akademii Krakowskiej (odnowienie 1400)"
  ],
  funFact:       "Przed koronacją był wielkim księciem litewskim i nosił imię Jogaila. Chrzest przyjął zaledwie 17 dni przed koronacją.",
  mnemonic:      "Jagiełło ↔ Grunwald 1410 — najsłynniejsza polska bitwa"
}
```

### Skrócony opis kontekstowy — wszystkie dynastie

**Piastowie (1025–1370)** — pierwsza polska dynastia; od Chrobrego przez rozbicie dzielnicowe do zjednoczenia przez Łokietka i złotego wieku Kazimierza Wielkiego.

**Andegawenowie (1370–1399)** — dwaj władcy z Węgier; Jadwiga — jedyna kobieta koronowana jako „król" (rex) Polski.

**Jagiellonowie (1386–1572)** — najdłuższa dynastia; unia z Litwą, złoty wiek kultury, ostatni Jagiellon (Zygmunt August) umarł bezpotomnie → wolna elekcja.

**Władcy elekcyjni (1573–1795)** — od Walezego do Poniatowskiego; szlachta wybiera króla; Sobieski pod Wiedniem; rozbiory kończą Rzeczpospolitą.

## Tryby quizu

Quiz powinien oferować **4 tryby gry**, żeby budować różne typy skojarzeń. Wspólna mechanika: streak 3× z rzędu do zaliczenia (jak w istniejących quizach).

### Tryb 1: Król → Data koronacji (wpisywanie)

Wyświetlamy imię króla i kontekst (dynastia, panowanie). Uczeń wpisuje rok koronacji.

```
┌─────────────────────────────────────┐
│    Władysław II Jagiełło            │
│    Jagiellonowie · 1386–1434        │
│                                     │
│    Rok koronacji: [____]  [Sprawdź] │
│                                     │
│    Seria: ●●○                       │
└─────────────────────────────────────┘
```

Po odpowiedzi — karta wiedzy (poprzednik, następca, wydarzenia, ciekawostka).

### Tryb 2: Data → Król (wybór z 4 opcji)

Wyświetlamy rok koronacji. Uczeń wybiera właściwego króla spośród 4 propozycji.

```
┌─────────────────────────────────────┐
│    Koronacja: 1025 rok              │
│                                     │
│    ○ Władysław Łokietek             │
│    ○ Bolesław I Chrobry             │
│    ○ Mieszko II Lambert             │
│    ○ Kazimierz III Wielki           │
└─────────────────────────────────────┘
```

Dystraktory dobierane inteligentnie — z tej samej dynastii lub bliskich dat.

### Tryb 3: Oś czasu (przeciągnij i upuść)

Losowane 5–8 królów — uczeń ustawia ich w kolejności chronologicznej na osi czasu (drag & drop).

```
┌─────────────────────────────────────────────────┐
│  Ułóż królów w kolejności koronacji:            │
│                                                 │
│  ╔══════════╗  ╔════════════╗  ╔══════════════╗ │
│  ║ Jagiełło ║  ║ Łokietek   ║  ║ Kaz. Wielki  ║ │
│  ╚══════════╝  ╚════════════╝  ╚══════════════╝ │
│                                                 │
│  [___] → [___] → [___] → [___] → [___]         │
│  wcześniej                        później       │
└─────────────────────────────────────────────────┘
```

### Tryb 4: Kontekst → Król (kto to?)

Wyświetlamy opis/wydarzenie. Uczeń rozpoznaje króla.

```
┌──────────────────────────────────────────────────┐
│  „Zastał Polskę drewnianą, a zostawił murowaną." │
│   Ostatni Piast na tronie. Panował 37 lat.       │
│                                                   │
│   Który to król? [________________________]       │
└──────────────────────────────────────────────────┘
```

## Karta wiedzy (wyświetlana po odpowiedzi)

Po każdej odpowiedzi (poprawnej lub błędnej) na ~3 sekundy pojawia się panel z kontekstem. Przy błędnej — dłużej (5 s), żeby uczeń miał czas przeczytać.

```
┌───────────────────────────────────────────────────────┐
│  ✓ Dobrze!  Władysław II Jagiełło — 4 marca 1386     │
│                                                       │
│  Dynastia:     Jagiellonowie                          │
│  Poprzednik:   Jadwiga Andegaweńska                   │
│  Następca:     Władysław III Warneńczyk               │
│                                                       │
│  ► Unia w Krewie (1385) — połączenie Polski i Litwy   │
│  ► Bitwa pod Grunwaldem (1410)                        │
│  ► Odnowienie Akademii Krakowskiej (1400)             │
│                                                       │

│     — nosił imię Jogaila.                             │

```

## Elementy budujące skojarzenia (metodyka)

| Technika | Realizacja w quizie |
|----------|---------------------|
| **Elaborative encoding** | Karta wiedzy z kontekstem po każdej odpowiedzi — uczeń powiązuje datę z wydarzeniami, dynastią, poprzednikiem/następcą |
| **Spaced repetition** | Streak 3× z rzędu; błędna odpowiedź resetuje serię → trudniejsze pytania wracają częściej |
| **Interleaving** | 4 tryby gry mieszają typ zadania (wpisywanie, wybór, sortowanie, rozpoznawanie) |
| **Retrieval practice** | Uczeń aktywnie przypomina sobie odpowiedź zamiast biernie czytać |
| **Dual coding** | Oś czasu (wizualna) + tekst; karta wiedzy łączy fakty z narracją |
| **Chunking** | Podział na dynastie — uczeń uczy się „paczkami" (Piastowie → Andegawenowie → Jagiellonowie → elekcyjni) |
| **Mnemoniki** | Pole `mnemonic` — krótkie zdanie-skojarzenie, np. „Jagiełło ↔ Grunwald 1410" |

## Filtrowanie po dynastii

Na ekranie startowym — możliwość wyboru zakresu:

- [ ] Piastowie (6 królów)
- [ ] Andegawenowie (2)
- [ ] Jagiellonowie (9 — w tym Jagiełło i Anna Jagiellonka)
- [ ] Władcy elekcyjni (11)
- [x] Wszyscy (28)

Dzięki temu uczeń może ćwiczyć mniejszymi porcjami (chunking).

## Technologia

- Samodzielny `index.html` (jak wszystkie quizy w projekcie)
- Zero zewnętrznych zależności (czysty HTML/CSS/JS)
- Dane wszystkich 28 królów jako tablica JS w pliku
- Drag & drop (tryb osi czasu) — natywne HTML5 Drag and Drop API
- Responsywny — działa na tablecie (typowe urządzenie w klasie)

## Kolejność implementacji

1. **Faza 1** — Tryb 1 (wpisywanie roku) + karta wiedzy + filtrowanie po dynastii
2. **Faza 2** — Tryb 2 (wybór z 4 opcji) + inteligentne dystraktory
3. **Faza 3** — Tryb 4 (kontekst → król)
4. **Faza 4** — Tryb 3 (oś czasu / drag & drop)
5. **Faza 5** — Ekran podsumowania z listą najtrudniejszych + kafelek na landing page
